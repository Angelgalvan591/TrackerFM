# Tracker FM
## Objetivo Principal
El proyecto Tracker FM
 busca crear una red social musical que permita a los usuarios registrar, organizar y compartir la música que escuchan diariamente. La idea es hacer del consumo musical una experiencia social, visual e interactiva, donde cada usuario pueda documentar su gusto musical y expresarlo a través de reseñas. Además, el objetivo es ayudar a los usuarios a descubrir nuevos perfiles de usuarios con intereses similares.
